# 协议与固件接口

V4.0 BIN 固定：

```text
version = 40
Header = 104
Node = 16
Segment = 24
Action = 22
```

固件运行侧应按 BIN 中 `vx/vy/wz` 作为 feedforward 目标读取，不把它们解释为速度上限。

正式 V4.0 完赛：

- `finish_mode = AT_FINAL_DROP`
- 最后一个放货 ARRIVAL 是唯一 `FINISH_ARM`
- 最终 `DROP_* STOP_AND_WAIT` 完成后，等待 `post_wait_ms`
- FIFO 为空
- 底盘仍停在最终 ARRIVAL

`SAFE_END` 与 `FINISH_CLEAR` 是保留位，正式输出必须为 0。
