# 错误排查

- `generation_mode mismatch`：Case 放错了 `manual/semi_auto/full_auto` 目录。
- `STALE`：配置、点、动作或依赖 leg 已改变；需要显式重新生成与验证。
- `target already exists`：迁移器拒绝覆盖，先处理冲突再执行。
- `no explicit generation_mode`：旧平铺文件无权威模式，迁移器不会猜测。
- `final action`：最后动作不是 DROP_* STOP_AND_WAIT 或未绑定最终 ARRIVAL。
- `leg locked/approved`：重算或清除需要加强确认。
- `collision` / `wheel rpm`：查看 Case 验证报告；降低速度不能修复真实几何碰撞。
