# 错误排查

常见问题：

- `legacy format`: 输入是 V3.x JSON/BIN，V4 不会静默迁移。
- `review.approved is false`: 工作输出尚未人工批准，不能 final export。
- `LOCK_CONFLICT`: 锁定候选已经不可用、hash 改变或依赖缺失。
- `missing or non-reusable leg`: leg 缺失、过期、preview、failed 或 hash/碰撞/动力学校验失败。
- `CRC mismatch`: BIN 文件损坏或不是当前编码器输出。
- `path_source mismatch`: Case 放错了 task/manual 目录。

错误处理原则：不静默修复损坏文件，不破坏旧 final，不留下半写文件。
