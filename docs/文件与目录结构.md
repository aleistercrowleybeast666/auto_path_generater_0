# 文件与目录结构

正式源码位于 `src/hjmb_pathgen/py_*`：`py_app` 负责组合入口，`py_ui` 只负责 Qt 表现，`py_domain` 是纯模型，`py_planning` 是几何/优化/动力学/碰撞，`py_services` 协调用例，`py_io` 负责 codec/布局/原子写入/迁移，`py_workers` 承担耗时任务，`py_utils` 放纯工具，`py_legacy` 只读旧格式。

运行项目目录：

```text
project.json
traj_id.csv
route_case_table.json
leg_library.json
presets/
cases/{manual,semi_auto,full_auto}/
bin/{manual,semi_auto,full_auto,final}/
portable/{manual,semi_auto,full_auto}/
reports/
drafts/
cache/
```

旧平铺、`manual_free`、`task_compiled` 均不是正式写入目标。迁移器按 Case 中的显式模式归档；无模式文件不会猜测，已有目标不会覆盖。
