# Phase 6 Curve Representation

The Phase 6 optimizer uses piecewise cubic Bezier curves.

Properties:

- endpoints are exact
- adjacent segments are C1 continuous
- samples are generated by arc-length resampling
- each sample carries tangent and curvature
- duplicate waypoints and degenerate derivatives are rejected

Initial guesses are deterministic:

- manual control points
- warm-start leg control points
- straight endpoint chord
- ordered gate-center polyline
- ordered gate-offset polylines

The stored `control_points` are library metadata. Dense executable geometry is
the `nodes` array after collision and dynamics validation.
