# 半自动模式

SEMI_AUTO Case 保存完整 8 个逻辑锚点：5 个取货侧公共姿态和 3 个当前 Case 放货姿态。用户可以编辑任务顺序、锚点 yaw、机械动作以及 `LOCKED_PASS`、`INITIAL_GUESS`、`OPTIMIZABLE` 辅助点。

生成时 8 个任务锚点不会移动。系统为相邻任务状态构造有向 leg，优化 Bezier 曲线和低速 yaw 窗口，复用 `leg_library.json`，拼接后再进行速度、碰撞、轮速、动作和 final-drop 验证。

FULL_AUTO 转换而来的 Case 保留 8 锚点、动作、原曲线路段引用和 `derived_from` hash；原全自动文件保持不变。
