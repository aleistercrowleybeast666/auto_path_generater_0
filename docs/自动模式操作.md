# 自动模式操作

自动模式使用 `traj_id.csv`、`route_case_table.json`、`project.json` 和 `leg_library.json`。

常用命令：

```powershell
python -m hjmb_pathgen.cli build-route-case-table --project <project>
python -m hjmb_pathgen.cli collect-unique-legs --project <project>
python -m hjmb_pathgen.cli optimize-missing-legs --project <project> --profile STANDARD
python -m hjmb_pathgen.cli generate-one --project <project> --traj-id 0
python -m hjmb_pathgen.cli generate-all --project <project>
python -m hjmb_pathgen.cli validate-all --project <project>
```

输出进入 `cases/task_compiled` 和 `bin/task_compiled`。不会自动写 `bin/final`。
