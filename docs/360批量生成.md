# 360 批量生成

`traj_id.csv` 是 360 映射唯一来源。

批量流程：

1. 校验 `traj_id.csv`。
2. 生成 `route_case_table.json`。
3. 收集 unique legs。
4. 优化缺失或过期 legs。
5. 生成 360 TASK_COMPILED Case 和 BIN。
6. 全量 validate。
7. 生成 golden manifest。
8. 需要交付时逐项批准并显式导出 final BIN。

同一输入、seed、profile、project 和算法版本应得到一致的 Case hash、BIN bytes 和 manifest hash。
