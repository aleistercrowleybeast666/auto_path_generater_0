# 用户操作手册

1. 新建或打开 V4 项目并确认项目公共姿态、物理对象、车辆和动作配置。
2. 选择 MANUAL、SEMI_AUTO 或 FULL_AUTO 以及 traj_id。
3. 在“路径编辑”页编辑允许的内容；编辑后 Case 仅标记 STALE。
4. 点击“生成/更新当前路径”，等待后台规划完成。
5. 点击“验证当前路径”，检查碰撞、运动学、轮速、动作和 final-drop。
6. 保存当前 Case，按模式导出 BIN；审批后才可“设为最终版本”。
7. FULL_AUTO 需要人工修改时先转为 SEMI_AUTO 副本。
8. 360 批量生成、路段优化和 clear leg 均在第二页执行。

旧工程使用菜单中的显式迁移，不要手工复制到新目录。详见 [三模式使用说明.md](三模式使用说明.md)。
