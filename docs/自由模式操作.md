# 自由模式操作

MANUAL_FREE Case 必须满足：

- `path_source = MANUAL_FREE`
- `selected_plan.route_family = MANUAL_FREE`
- `review.detached_from_library = true`
- `review.manual_override = true`
- `review.override_reason` 非空
- `manual_path` 合法

输出进入 `cases/manual_free` 和 `bin/manual_free`，可与同一个 `traj_id` 的自动模式共存。
