# 三模式使用说明

同一 traj_id 可以同时拥有 MANUAL、SEMI_AUTO、FULL_AUTO 三份 Case。切换模式只加载对应目录，不复制、不覆盖。

- MANUAL：先编辑点列和动作，再点击“生成/更新当前路径”。
- SEMI_AUTO：编辑 8 个锚点、yaw、动作及辅助点，再显式生成优化路段。
- FULL_AUTO：选择 traj_id 后生成当前 ID，或在第二页生成全部 360。

任何编辑只会把 Case 标记为 `STALE`。生成、验证、导出 BIN 和设为 final 都是独立显式操作。FULL_AUTO 不能直接编辑；使用“转为半自动编辑”创建带来源 hash 的副本。

模式文件分别位于 `cases/<mode>`、`bin/<mode>` 和 `portable/<mode>`。通过完整验证、非 STALE 且已审批的 Case 才能发布到 `bin/final`。
