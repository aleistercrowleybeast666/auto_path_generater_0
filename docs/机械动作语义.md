# 机械动作语义

V4.0 动作 FIFO 按 `action_seq` 执行。

- `STOP_AND_WAIT`: 到达绑定 ARRIVAL 并停稳后启动，等待模块 `DONE`，再等待 `post_wait_ms`。
- `ASYNC`: 到达 FIFO 队首后启动，不带路径触发位置，但 FIFO 仍等待其完成后再执行后续动作。
- `KINEMATIC`: 不带用户窗口和 expiry；`check_start_s_mm` 由最终拼装轨迹扫描生成，满足 speed/accel/wz/beta/stable_time。

最终动作必须是 `DROP_1`、`DROP_2`、`DROP_3`、`DROP_12` 或 `DROP_23`，模式为 `STOP_AND_WAIT`，绑定最终 `FINISH_ARM` ARRIVAL。
