# 项目配置说明

`project.json` 是共享人工配置，包含：

- nominal field: 4000 x 2000 mm
- 10 个固定站点
- 车辆 footprint 与四轮参数
- 场地障碍物和边界
- dynamics
- unload profiles
- topology profiles
- action profiles
- planner profiles
- start/arrival/final check policy

示例工程是合成配置，只用于软件流程复现。真实比赛前必须重新测量站点、障碍物、车辆半径、轮半径、轮心旋转半径、动作耗时和安全余量。
